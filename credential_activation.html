📨 1. Create SendGrid Account

Free tier: 100 emails/day, no credit card required.

Tap the button below to open the SendGrid signup page. Use samuelang88@gmail.com. When asked for a credit card — click "Skip".

Open SendGrid Signup signup.sendgrid.com — takes ~2 minutes
After signing up, check your Gmail inbox for the verification email and click the link before proceeding.

🔑 2. Create API Key

One key unlocks the SMTP relay. Copy it now — shown only once.

  1. In the SendGrid dashboard, go to Settings → API Keys
  2. Click "Create API Key", name it himalaya-smtp
  3. Select "Full Access" → "Create & View"
  4. COPY THE KEY NOW — it starts with SG.

⚙️ 3. Insert API Key into Config

Paste this into your Himalaya config — then replace the placeholder with your real key.

Open your config file:

$ nano ~/.config/himalaya/config.toml

Add or find the [accounts.sendgrid] section and paste this:

[accounts.sendgrid] default = true email = 'samuelang88@gmail.com' sender = 'samuelang88@gmail.com' smtp.host = 'smtp.sendgrid.net' smtp.port = 587 smtp.login = 'apikey' smtp.password = 'YOUR_SENDGRID_API_KEY_HERE'
Copied!
Replace YOUR_SENDGRID_API_KEY_HERE with the SG.xxxxx.xxxxx key you copied in Step 2. Keep the single quotes.

🚀 4. Fire the 34 Emails

One command. All 34 Day 0 cold emails. Review before each send.

Once your API key is in the config, run:

$ bash ~/drone-design/outreach/batch_send_emails.sh --account sendgrid

The script will prompt you before each send so you can review. You can also test with a single email first:

$ echo -e "From: samuelang88@gmail.com\nSubject: test\n\nSendGrid works!" | himalaya --account sendgrid message send
